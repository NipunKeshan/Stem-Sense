Website Type

This is a data analytics dashboard website for an IoT plant monitoring system.

The design should look like a professional IoT analytics platform used in smart agriculture systems.

Pages Required

Create the following pages:

1️⃣ Dashboard Overview

Displays overall plant health and system summary.

Components:

Soil moisture status card

Latest sensor readings

Quick irrigation status

System connectivity status

Alerts summary

2️⃣ Soil Moisture Monitoring Page (Main Page)

This page focuses on irrigation decision support.

Components:

Real-Time Moisture Card

Large soil moisture percentage

Status label: Dry / Moderate / Optimal

Circular gauge visualization

Moisture Trend Chart

Line chart showing soil moisture over time

X axis: time

Y axis: moisture percentage

Include threshold indicator for minimum safe moisture

Irrigation Control Panel

Pump status indicator

Start watering button

Stop watering button

Last irrigation time

Moisture Alerts

Notification when soil becomes too dry

Warning messages

3️⃣ Environmental Monitoring Page

Shows other sensor data.

Components:

Temperature card

Humidity card

Environmental trend charts

Environmental status indicators

4️⃣ Motion Detection / Disturbance Page

Displays environmental disturbances.

Components:

Motion detection alerts

Event timeline chart

Disturbance history log

5️⃣ System Health Page

Displays device and sensor status.

Components:

Sensor connectivity status

Device uptime

Error logs

System performance indicators

Layout Structure

The website should include:

Top Header

System title: Smart Plant Monitoring Dashboard

Notification icon

Left Sidebar Navigation
Menu items:

Dashboard

Soil Moisture

Environment

System Health

Dashboard Design Style

The design should look like a modern IoT analytics dashboard.

Use card-based layout with:

rounded containers

soft shadows

clean spacing

clear data visualization

Color Palette

Use natural plant related colors.

Primary color
Dark green (#1B5E20)

Secondary color
Leaf green (#2E7D32)

Highlight color
Light green (#A5D6A7)

Alert colors
Red (#E53935) for critical alerts
Orange (#FB8C00) for warnings

Background
Light grey (#F5F5F5)

Cards
White (#FFFFFF)

Data Visualization

Include the following charts:

Line chart for soil moisture trends
Bar chart for daily irrigation usage
Gauge chart for real-time moisture level
Timeline chart for motion events

Charts should include hover tooltips and legends.

Interactive Features

The website should support:

Real-time sensor updates
Data filtering by time (24 hours / 7 days / 30 days)
Manual irrigation control
Alert notifications
Responsive design for desktop and tablet

Data Outputs

Display:

Soil moisture percentage
Moisture trend patterns
Pump on/off status
Irrigation history
Sensor status
Environmental readings

Design Goal

The interface should allow users to quickly understand plant hydration conditions and make irrigation decisions with minimal effort.

The dashboard must prioritize:

visual clarity

fast interpretation

real-time monitoring

simple interactions.
